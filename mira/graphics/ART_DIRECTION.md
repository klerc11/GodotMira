# LIGHT TRANSMISSION - ART DIRECTION

## Core Identity
- MIRA is a dead transmission civilization built to route light across impossible distances.
- The beam is not decoration. It is gameplay, navigation, pacing, composition, and identity.
- Rule: **the beam is the detail**.

## Primary Rule
- Keep world visuals restrained so transmission reads first at all times.
- Saturation belongs to:
  - transmission
  - activation
  - gameplay state
  - energy flow
- Saturation does **not** belong to:
  - base architecture
  - random props
  - decorative neon
  - uncontrolled lighting

## World Feel
- Dormant, cold, silent, engineered, monolithic, abandoned.
- The city is a gigantic dead machine with residual energy.
- Emotional target: carrying the final living transmission through silent infrastructure.

## Architectural Language
- Industrial / infrastructural / modular / brutalist.
- Large forms first:
  - slabs, pylons, relay gates, conduit trenches, transfer halls, transit corridors.
- Avoid noisy micro-detail and decorative clutter.
- Every form should imply function in a transmission system.

## Readability Rules
- Gameplay readability beats visual complexity.
- At sprint speed, players must read route, geometry intent, and beam direction.
- Use negative space intentionally:
  - emptiness amplifies scale
  - emptiness amplifies loneliness
  - emptiness amplifies beam visibility

## Movement + Architecture
- Geometry should support flow:
  - directional corridors
  - momentum lanes
  - vertical transfer paths
  - compression/exposure spaces
  - long uninterrupted traversal lines

## Color Philosophy

### Structural Palette
- Graphite Black: `#16181C`
- Blue-Gray Steel: `#2A3138`
- Wet Concrete Gray: `#4B545D`

### Atmosphere-Only Purple
- Deep Violet Fog: `#312B42`
- Muted Shadow Purple: `#4A4360`

Purple stays in atmosphere, haze, shadow tint, and distant depth.  
Purple should not dominate architecture.

### Beam Palette
- Primary Beam Mint: `#8FFFD1`
- Beam Core White: `#EFFFF8`

### Secondary Accent (sparse)
- Relay Amber: `#D6A45A`
- Faint Signal Red: `#9B4D58`

Use red only for failure/instability/emergency/dead-system moments.

## Character Direction
- Character stays visually secondary to beam and environment scale.
- Silhouette clarity > outfit detail.
- Character palette:
  - base: `#1C1E22`
  - muted accent purple: `#6E5A8A`
  - beam reflection tint near active transmission: `#A5FFE0`

## Lighting Philosophy
- Core contrast: dead darkness vs active transmission.
- Do not globally awaken the city.
- Prefer localized activation:
  - conduit flickers
  - relay ignition
  - nearby reflective response
  - then decay back to silence

## Atmosphere
- Cold, heavy, humid, quiet.
- Use fog/haze/soft volumetrics sparingly for scale and depth.
- Avoid over-fogging, over-blooming, or particle noise that reduces readability.

## Do / Do Not

### Do
- simplify
- use giant forms
- preserve negative space
- keep silhouettes clean
- make structure feel functional
- let the beam dominate composition

### Do Not
- clutter spaces
- over-detail geometry
- overuse neon
- overpower beam with environment color
- use random sci-fi props as identity
